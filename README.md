## ğŸ§© Platform Architecture

- **Backend:** Kotlin + Ktor  
- **Frontend:** Kotlin Multiplatform + Compose Multiplatform  
- **Mobile Apps:** Android & iOS (native via KMP)  
- **Database:** PostgreSQL / Firebase (tenant-based structure)
